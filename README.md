## babel-import-plugin

### 安装
```bash
npm i @jomsou/babel-import-plugin -D
```

### 使用

#### .babelrc配置
```js
{
  "plugins": [
    "czq-import"
  ]
}
```

#### webpack配置
```js
rules: [
    {
      test: /\.js$/, use: {
      loader: 'babel-loader',
      options: {
        presets: [
            'env',
            'react'
        ],
        plugins: [
            'czq-import'
        ]
      }
    }, exclude: /node_modules/}
  ]
}
```
